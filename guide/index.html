WebTau

WebTau (Web Test automation) - concise and expressive way to create REST API and Web UI tests. package scenarios.rest import static com.twosigma.webtau.WebTauGroovyDsl.* scenario("simple get") { http.get("/weather") { temperature.shouldBe < 100 } } json { "temperature": 88 } Tests can be written in any JVM language with language specific syntactic sugar where applicable. package scenarios.ui import static com.twosigma.webtau.WebTauGroovyDsl.* import static pages.Pages.* scenario("search by specific query") { search.submit("search this") search.numberOfResults.should == 2 }
