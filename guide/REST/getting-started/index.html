Installation

Download and unzip webtau. Add it to your PATH.

Bare Minimum

examples/rest/restGet.groovy
package rest

import static com.twosigma.webtau.WebTauGroovyDsl.*

scenario("simple get") {
    http.get("/weather") {
        temperature.should == 88
    }
}
To run test, navigate to examples dir and
$ 
webtau rest/restGet.groovy --url=https://my-server
Note:using package and import is optional and is mainly for IDE auto completion. Imports will be added implicitly during command line run.

Config File

Url parameter can be moved to a webtau.cfg file.
examples/rest/webtau.cfg
url = "http://localhost:8180"
Specify multiple environments to streamline test execution.
