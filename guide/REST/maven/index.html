Dependency

You can use maven to add webtau as a dependency to you project (for autocompletion or to use with JUnit like runners).
GroovyJava
<dependency>
    <groupId>com.twosigma.webtau</groupId>
    <artifactId>webtau-groovy</artifactId>
    <version>0.29</version>
</dependency>

Plugin

Use maven plugin to run pure groovy tests as part of your build.
<plugin>
    <groupId>com.twosigma.webtau</groupId>
    <artifactId>webtau-maven-plugin</artifactId>
    <version>0.29</version>
    <executions>
        <execution>
            <phase>test</phase>
            <goals>
                <goal>run</goal>
            </goals>
        </execution>
    </executions>
    <configuration>
        <workingDir>${project.basedir}/src/main/groovy</workingDir>
        <env>integration</env>
        <url>http://optional-base-url</url>
        <tests>
            <directory>${project.basedir}/src/main/groovy</directory>
            <includes>
                <include>scenarios/*.groovy</include>
            </includes>
        </tests>
    </configuration>
</plugin>
Use env to specify environment to use. Alternatively you can use url to override base url.
