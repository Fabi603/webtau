Contain

Use contain matcher to test scenarios like search or list of recently created entries. This way you don't have to assume an existing state of your backend under test.Given the response, we want to make sure there is an entry with a specified firstName and lastName . package scenarios.rest.springboot import static com.twosigma.webtau.WebTauGroovyDsl.* scenario("list Customers and assert that it contains a specified entry") { http.get("/customers/search/findAllByOrderByLastName") { _embedded.customers.should contain([firstName: 'FN1', lastName: 'LN1']) } }
