Base URL

Robust tests don't specify the full URL of an application under test. Instead you only pass a relative URL to functions like open.
import static com.twosigma.webtau.WebTauGroovyDsl.*

open("/relative-url")
Define base URL portion either inside a webtau.cfg file
url = "http://localhost:8180"
userName = 'testUser'
environments {
   dev {
       url = "http://dev.host:8080"
   }
}
or pass as a command line argument --url=http://...

Timeouts

Default timeout in milliseconds for waitTo and waitToNot
waitTimeout = 25000

--waitTimeout=25000

Window Size

Browser window size can be set using windowWidth and windowHeight
url = "http://localhost:8180"

windowWidth = 1280
windowHeight = 800

Documentation Artifacts

By default all generated documentation artifacts (e.g. screenshots) are created in the current directory. To override
url = "http://localhost:8180"

docPath = "screenshots"
