Timeouts

Default timeout in milliseconds for waitTo and waitToNot
waitTimeout = 25000

--waitTimeout=25000

Window Size

Browser window size can be set using windowWidth and windowHeight
url = "http://localhost:8180"

windowWidth = 1280
windowHeight = 800

Documentation Artifacts

By default all generated documentation artifacts (e.g. screenshots) are created in the current directory. To override
url = "http://localhost:8180"

docPath = "screenshots"
