Installation

Webtau provides cli. module to run and validate command line tools. Combine it with other modules to create powerful tests: use cli to run a tool, and http , graphql , browser , or db to validate the outcome.If you use Webtau Groovy runner, the cli. module is available as part of convenient static import: import static org.testingisdocumenting.webtau.WebTauGroovyDsl.* Note: Import is optional and is added for IDEs benefits.

Groovy Runner Example

scenario("example of ls run") { cli.run('ls') { output.should contain('scripts') } } webtau basicLs.groovy examples pom.xml pom.xml.versionsBackup scripts src target test-expectations
